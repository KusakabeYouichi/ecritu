索引五十音順索引

機能名・キー名・記号から該当ページの節へ直接ジャンプできます。記号と英数字ではじまる項目は先頭にまとめています。

記号・英数字

  • 、キー(読点キー — 。?!・)
  • ()ラップモード
  • …(三点リーダ)
  • 〜(波ダッシュ)・ー(長音)
  • ⌫(削除キー)
  • ☺︎キー(かな左下)
  • 2段フリック(括弧)
  • 3×3+わ配列(かな)
  • 5×2配列(かな)
  • Apple方向 / écritu方向
  • AZERTY配列
  • calculette(数字配列)
  • clavier(数字配列)
  • Drapeaux(国旗)
  • édition 番号
  • QWERTY配列
  • téléphone(数字配列)

あ行

  • アクセントカラー(émeraude / tuilé)
  • アクセント文字(à é ç などの長押し)
  • あひるアイコン(修飾キー)
  • 英字の入力
  • 英字の配列(設定)
  • 英単語サジェスト
  • 絵文字パネル
  • 絵文字・顔文字を変換候補に
  • オープンソースライセンス
  • 温度単位(℃ / ℉)

か行

  • ガイド文字表示(オフ / 4方向 / 下)
  • 学習語彙
  • 学習語彙の全削除とリセットの違い
  • 確定(改行キー)
  • 括弧の入力
  • カタカナで確定
  • カタカナ入力モード(ひらがな⇄カタカナ)
  • かな識別の学習(かなのまま確定を覚える)
  • かな修飾(前置 / 後置)
  • かな配列(5×2 / 3×3+わ)
  • 顔文字パネル
  • 顔文字の検索
  • 基本記号の並び順(ASCII / EBCDIC / ANSI)
  • キーボードの追加(有効化)
  • キーボードの切り替え
  • 記号パネル(7カテゴリ)
  • 旧仮名遣い候補
  • 句読点入力時の自動確定候補
  • 国旗(長押しで正式名称)
  • 候補バーの読み方
  • 小文字の入力(っ・ゃ など)

さ行

  • 削除キーリピート(速度設定)
  • 三点リーダ(…)
  • シフトキー(タップ / ロック)
  • 自動確定(アイドル)
  • 修飾キー(小・゛・゜)
  • ショートカット語彙(定型文)
  • 状態カプセル(鉛筆=未確定 / 循環矢印=変換中)
  • 診断ログ(キーボード診断)
  • 数字の入力
  • 数字の配列(設定)
  • 送信したら文字が消えた

た行

  • 第1語彙・第2語彙(内蔵辞書)
  • 濁点・半濁点
  • タブ文字(空白キー上フリック)
  • 追加語彙(単語の手動登録)
  • 通貨記号(長押しで通貨コード)
  • テーマカラー(bleu / sakura)
  • 導入手順

な行

  • 中黒(・)
  • 波ダッシュ(〜)

は行

  • パスワード欄でキーボードが出ない
  • フリック入力のきほん
  • フリック方向(écritu / Apple)
  • フルアクセスの許可
  • 変換(空白キー)と候補送り
  • 変換候補が出ない・少ない

ま行

  • 未確定のまま送信すると消える
  • キーボード全体の見取り図

や行

  • やキー(『』と2段フリック)
  • ユーザ辞書(iOS本体)の単語を使う
  • 抑制語彙(出したくない候補)
  • 横画面のレイアウト
  • 読みから絵文字・顔文字に変換

ら行

  • 連絡先の名前を候補に

わ行

  • わキー(〜・ー・を・ん)
