第5章絵文字・顔文字・記号パネル

écritu には、絵文字・顔文字・特殊記号のための 3 つのパネルがあります。どれもかなキーボードの左下からワンアクションで開けて、国旗や通貨記号は「長押しで名前が分かる」ちょっとした仕掛けつきです。

開き方 — ☺︎ キー

かなキーボード左下の ☺︎ キーが 3 つのパネルの入口です。

操作開くパネル
タップ絵文字パネル(☺︎)
右フリック顔文字パネル(^_^)
上フリック記号パネル(⌘)

タップ・右フリック・上フリックにどのパネルを割り当てるかは設定で入れ替えられます(第7章)。かなキーボードに戻るには、各パネルの左下にある あい ボタンを押してください。どのパネルにも右下に ⌫ があるので、打ち間違いはその場で消せます。

絵文字パネル

絵文字パネルは、上下にスクロールする絵文字グリッドと、下段のカテゴリボタンでできています。カテゴリボタンをタップすると、そのカテゴリの先頭にジャンプします。カテゴリは次の 8 つで、この順に並んでいます。

ボタンカテゴリ
😀顔・人物
🍷食べ物・飲み物
🦆動物・自然
🏀行動・スポーツ
💡オブジェクト
🚗乗り物・場所
🇫🇷国旗(Drapeaux)
❤️シンボル

カテゴリの中はさらに細かいグループ(顔/手/人物…など)に分かれていて、グループの間には区切り線が入ります。絵文字をタップするとそのまま入力されます。

国旗(Drapeaux) — 長押しで正式名称

国旗カテゴリは 国 → 海外領土・属領 → その他の旗 の 3 グループに区切り線で分かれています。先頭はもちろん 🇫🇷 です。

旗のデザインだけでは国名が分かりにくいもの — そんなときは旗を長押ししてください。押している間、ISO 3166 の正式名称(フランス語)がバブルで表示されます。たとえば 🇫🇷 は「France」、🇩🇪 は「Allemagne」です。

ISO の国・地域ではない旗(ウェールズ 🏴󠁧󠁢󠁷󠁬󠁳󠁿、カナリア諸島 🇮🇨、コソボ 🇽🇰、サルク島 🇨🇶、EU 🇪🇺、交差した日の丸 🎌)は、通常の黒いバブルではなく青いバブルで名称が表示され、正式な国名でないことが分かるようになっています。

ちなみに:Drapeaux(ドラポー)はフランス語で「旗」。écritu のフランス語風ネーミングの 1 つです(第1章)。

記号パネル

記号パネルには 7 つのカテゴリがあります。使い方は絵文字パネルと同じで、下段のカテゴリボタンで切り替え、記号をタップして入力します。

『』括弧・引用符()[]{}【】〔〕〈〉《》、“” ‘’ «» など
€通貨€ $ ¢ £ ¥ ₩ ₹ など 24 種+₿ と暗号資産の代替表記

基本記号の並び順

基本記号カテゴリの並び順は、設定で ASCII(文字コード順・既定)/EBCDIC/ANSI(米国キーボードの打鍵順)から選べます(第7章)。どの並びでも、末尾の図形記号グループは共通です。

通貨は長押しでコードが分かる

通貨カテゴリの記号を長押しすると、押している間バブルで名前が表示されます。

  • 通常の通貨記号 — ISO 4217 の通貨コード。€ なら「EUR」、₹ なら「INR」です。
  • ₿ と暗号資産の代替表記 — ティッカーシンボル(₿=BTC、Ξ=ETH など)。
  • 補助単位の ¢ と ₰ — ISO コードを持たないため、緑のバブルで名称(cent/Pfennig)が表示されます。

温度単位は iOS 設定に連動

単位カテゴリの先頭は温度記号です。iOS 本体の温度単位設定が摂氏なら「℃ ℉」の順、華氏なら「℉ ℃」の順に並び、カテゴリボタンのアイコンもそれに合わせて ℃/℉ に切り替わります。

顔文字パネル

顔文字パネルは、下段のカテゴリボタンで分類を切り替えて使います。先頭の 3 つは特別なカテゴリです。

ボタンカテゴリ内容
⚡️ショートカット自分で登録したショートカット語彙(第6章)がここに並びます。よく使う顔文字や定型文の置き場に。
🙂基本標準セットの顔文字。
🔎検索読みから顔文字を探せます(下記)。

その後ろに、気分・場面ごとの分類が続きます:笑顔/かわいい/照れ/焦り/しょぼん/悲しい/怒り/驚き/くそねみ(眠い)/挨拶/ラブ/激しい/うごき/キモい/キャラ/特殊/ライン(区切り線)。カテゴリ名はフランス語+日本語の併記です。

読みで検索 — 3 ステップ

検索カテゴリ(🔎)を開くと、パネル上部に手順が表示されます:「1) 上の文字を選ぶ 2) 読みを選ぶ 3) 下の顔文字をタップ」。

  1. 横に並んだ五十音の見出し(「全」を選べば全部)から、読みの頭文字を選びます。
  2. 表示された読みの一覧(「えがお」「おこ」など)から 1 つ選びます。「よみ: えがお」と表示され、「読みを選び直す」ボタンでやり直せます。
  3. 「候補をタップして入力」の下に並んだ顔文字をタップすると入力されます。

読みから変換 — パネルを開かない近道

実は、パネルを開かなくても顔文字・絵文字は入力できます。かなのまま「えがお」「ねこ」「くるま」などと打つと、変換候補の中に顔文字や絵文字が混ざって表示されます。^_^ や 🚗 が普通の変換と同じ操作で確定できるので、慣れるとこちらの方が速いはずです。

絵文字の読みは Unicode CLDR の日本語データをもとに整備されているので、「らっこ」🦦 のような新しめの絵文字も読みから出せます。候補に絵文字・顔文字を混ぜるかどうかは、それぞれ設定でオフにできます(第7章)。
